# Dependencies

```
$ sudo apt install clang libclang1
```

# Features

打开文件自动进入 Less 模式（只读模式），类似 Vim，有的时候不进行编辑操作，在 Less 下可以快速阅读代码。

# 快捷键

## 文本模式
功能名 | 快捷键
:--- | ---
Less | C-l
注释隐藏 | M-m

## Dired (文件管理器)
功能名 | 快捷键
:--- | ---
新建文件 | E
搜索名称 | S
上一级目录 | q
打开文件或目录 | e
文件大小 | s
重命名 | n

## Speedbar (导航栏)
功能名 | 快捷键
:--- | ---
弹出 | M-j
展开当前行 | x
收缩当前行 | z
刷新 | g
上一级目录 | q
删除 | d
重命名 | r
创建目录 | m
